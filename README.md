# Legacy Python

Base images for legacy Python versions 0.9.1, 1.x, 2.x, and 3.x versions. More recent versions of Python 3 are readily available through the official [Docker Python image](https://hub.docker.com/_/python) and version managements tools like [pyenv](https://github.com/pyenv/pyenv) or [uv](https://github.com/astral-sh/uv).
